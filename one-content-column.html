Get started

Yasu API provides programmatic access to many different random images, which can be then used for a number of things.

Rate Limit

Yasu API uses a 50 requests per minute rate-limit, so don't go around spamming the API.

get images

To get images you need to make a GET request to the following url :
http://replaceme/{endpoint}


ENDPOINTS

Field Type Description
cars GET Random Car image
sofaygo GET Random SoFaygo image
juicewrld GET Random Juice WRLD image
oliviarodrigo GET Random Olivia Rodrigo image
yeat GET Random YEAT image
travis GET Random Travis Scott image
kanye GET Random Kanye West image
capybara GET Random Capybara image
zzz GET Random Zzz. image

Errors

Yasu API uses the following error codes:

Error Code Meaning
404 Misspelled or Invalid endpoint
429 You are being rate-limited
